DETAILS
The Vayu Timer Support module provides a timer available mask for each core that uses dmtimers - A15, DSP core 0 & DSP core 1.
dmtimer Timer module's anyMask should not be modified directly. All changes to anyMask must be made via the Vayu TimerSupport module's timer available mask. Care must be taken to not overlap the available masks for the different cores.
